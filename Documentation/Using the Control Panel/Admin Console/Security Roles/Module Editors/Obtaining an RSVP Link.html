Obtaining an RSVP Link

How to obtain an RSVP link using the Security Roles module. Once you have created a security role with an RSVP Code an RSVP Link will be generated for that role. Users can subscribe to the role simply by clicking on the link or going to the URL.

  1. Navigate to Admin > Security Roles - OR - Go to a Security Roles module.
  2. If the required role is not displayed, at Filter By Role Group select the [Role Group Name] associated with the role, or select [All Roles]< All >.
  3. Click the Edit button beside the required role.
  4. Select the Advanced Settings tab.
  5. At RSVP Link, copy the link.
