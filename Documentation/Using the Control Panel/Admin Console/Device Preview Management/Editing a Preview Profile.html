Editing a Preview Profile

How to edit one or more fields of a preview profile that is listed in the Device Preview Management module.

  1. Navigate to Admin > Advanced Settings > Device Preview Management.
  2. Click the Edit button beside the Device Profile to be edited. This displays the selected record in the edit fields at the top of the module.
  3. Edit one or more fields as required.

  1. Click the Save button to save your changes.
