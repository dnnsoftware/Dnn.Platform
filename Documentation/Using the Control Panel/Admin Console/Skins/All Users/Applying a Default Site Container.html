Applying a Default Site Container

How to apply a container as the default container for all modules on a site. This setting does not override containers set for individual modules on the module settings page.

  1. Navigate to Admin > Advanced Settings > Skins and go to the Skin Editor module - OR - Go to a Skins module.
  2. Locate and optionally preview the required container. See "Previewing a Container Package"
  3. At Apply To, select one or both of the following options:
    • Website: to apply the container to all site pages.
    • Admin: to apply the container to all Admin Console pages.
  4. Click the Apply button below the chosen container.
