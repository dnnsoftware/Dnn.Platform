Parsing a Skin Package

How to parse a skin package. This task can only be performed by Hosts.

  1. Navigate to Admin > Advanced Settings > Skins and go to the Skin Editor module - OR - Go to a Skins module.
  2. At Skins, select a skin package from the drop down list.
  3. At Parse Options, select from the following options:
    • Localized: This option includes the full path.
    • Portable: This option does not include the full path.
  4. Click the Parse Skin Package button. A detailed report is displayed.
