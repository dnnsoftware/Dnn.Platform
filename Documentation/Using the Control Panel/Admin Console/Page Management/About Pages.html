About Pages

DNN sites are created by adding one or more pages and then adding modules that display content onto those pages. Administrator, SuperUsers and user with Add permissions can add pages. Pages can be added by clicking Add New Page in the Pages menu of the ControlBar or via the Admin > Page Management page, See "About the Pages (Tabs) module".
