Configuring Google Analytics

How to configure either the Google Analytics module.

You must have an existing Google Analytics account. Visit the Google Analytics website at http://www.google.com/analytics and click the Create an Account button.

  1. Navigate to Admin > Advanced Settings > Google Analytics - OR - Go to a Google Analytics module.
  2. In the Tracking ID text box, enter the website tracking ID that you obtained from the welcome message sent by Google when you signed up for an account.
  3. Optional. In the URL Parameter text box, enter the JavaScript code fragment that will be passed as a parameter to the page tracker. This is used to customize the value tracked in Google Analytics instead of the default location.
