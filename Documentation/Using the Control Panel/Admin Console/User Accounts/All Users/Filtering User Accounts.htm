Filtering User Accounts

The following filters can be applied user account records

  1. Navigate to Admin > User Accounts - OR - Go to a User Accounts module.

  2. Select from these options:
    • Click the All link to view all user accounts. By default, the first ten (10) accounts are listed in alphabetical order by username.
    • Click the Deleted link to view users whose account has been deleted and have had their access removed however their user account information has not yet been removed.
    • Click the Online link to view users who are currently logged in to this site. This filter is integrated with the Users Online module which must be enabled by a SuperUser for this filter to work. See "Configuring Users Online Settings"
    • Click on the linked [letter of the alphabet] which is the first letter of the persons user name. This displays all matching user accounts is alphabetical order.
    • Click the Unauthorized link to only display user accounts that have been unauthorized. These records display the Unchecked icon in the Authorized column. By default, the first ten ( 10 ) unauthorized user accounts are listed in alphabetical order by username.
