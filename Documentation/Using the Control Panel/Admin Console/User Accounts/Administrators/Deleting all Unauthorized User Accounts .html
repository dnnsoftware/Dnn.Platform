Deleting all Unauthorized User Accounts

How to permanently delete all unauthorized user accounts using the User Accounts module.

  1. Navigate to Admin > User Accounts - OR - Go to a User Accounts module.
  2. Optional. Click the Unauthorized link to view a list of unauthorized users before deleting them.
  3. Click the Delete Unauthorized Users link at the base of the module. This displays the message "Are You Sure You Wish To Delete These Items?"

  1. Click the Yes button to confirm deletion.
