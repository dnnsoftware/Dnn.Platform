Editing a User Account

How to edit the details of a user's account using the User Accounts module. Note: The user name field cannot be edited.

  1. Navigate to Admin > User Accounts - OR - Go to a User Accounts module.
  2. Find the user to be edited using a filter or by searching.
  3. Click the Edit button beside their record. This opens the Edit User Accounts page.
  4. Edit one or more fields as required. Editable fields are the user's first name, last name, display name and email address.
