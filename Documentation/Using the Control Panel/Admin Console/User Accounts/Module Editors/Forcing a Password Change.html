Forcing a Password Change

How to force a user to change their password next time they login to the site using the User Accounts module.

  1. Navigate to Admin > User Accounts - OR - Go to a User Accounts module.
  2. Find the required user using a filter or by searching.
  3. Click the Edit button beside their user account.
  4. Select the Manage Acount tab.
  5. Click the Force Password Change button. This removes the Force Password Change link; sets the Update Password field to True; and displays the "User must update password on next login" message.
