Editing User Entered Tags

How to edit tags which have been created by users.

  1. Navigate to Admin >  Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit button beside the pre-existing entry named "Tags". This opens the Edit Vocabulary page.

  1. In the Terms list, select the tag to be edited. This opens the Edit Term page.
  2. In the Edit Term section, edit one or more fields:
    1. In the Name text box, edit the name of this term.
    2. In the Description text box, edit the description of this term.

  1. Click the Update button.
  2. Repeat Steps 3-5 to edit additional tags.
