Deleting User Entered Tags

How to delete tags that have been created by users.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. In the List of Vocabularies, click the Edit button beside the preexisting entry named "Tags". This opens the Edit Vocabulary page.
  3. In the Terms list, select the tag to be edited. This displays the Edit Term section.
  4. Click the Delete button.
  5. Repeat Steps 3-4 to delete additional tags.
