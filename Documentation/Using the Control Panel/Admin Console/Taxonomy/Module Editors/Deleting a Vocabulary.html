Deleting a Vocabulary

How to delete a vocabulary from the Taxonomy Manager module. Editors cannot delete vocabularies created by Administrators or SuperUsers. Similarly, Administrators cannot delete vocabularies created by SuperUsers.

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit button beside the vocabulary to be deleted. This opens the Edit Vocabulary page.
  3. Click the Delete button.
