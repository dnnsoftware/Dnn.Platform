Adding Additional Terms (Hierarchical Vocabulary)

How to add terms to a hierarchical vocabulary using the Taxonomy Manager module. Important. The first term for a vocabulary requires a different process. See "Adding the First Term (Hierarchical Vocabulary)"

  1. Navigate to Admin > Advanced Settings > Taxonomy - OR - Go to a Taxonomy Manager module.
  2. Click the Edit button beside the required vocabulary. This opens the Edit Vocabulary page.
  3. Click the Add New button. This displays the Create New Term section.
  4. In the Create New Term section, complete these fields:
    1. In the Name text box, enter a name for this term.
    2. In the Description text box, enter a description for this term.
    3. At Parent Term, select the parent term for this term.
  5. Click the Save button. The new term is now displayed in the Terms hierarchical list.
  6. Repeat Steps 3-5 to add additional terms.
