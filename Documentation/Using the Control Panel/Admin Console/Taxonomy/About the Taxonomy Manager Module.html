About the Taxonomy Manager

The Taxonomy Manager, located on the Admin > Advanced Settings > Taxonomy page, allows you to create and manage tags which are used to classify site content. Tags can be associated with pages (See "Page Details Settings for Existing Pages") and modules. Depending on the skin used on your site, users may be able associate content with existing tags or create their own tags that can then be managed using the Taxonomy Manager. The Taxonomy Manager can be added to site pages.

Authenticated users who can view the module can see the list of existing vocabularies. Editors and Administrators can create and manage site specific vocabularies. SuperUsers can create and manage application wide vocabularies that are available to all sites.
