Viewing All Vendor Accounts

How to view all vendor accounts on the Vendors module.

  1. Navigate to Admin > Advanced Settings >  Vendors.
  2. Click the All link. This displays the first ten (10) matching records listed alphabetically by name. If there are more than ten matching records, then the Pager Control is displayed allowing you to access the other records.
