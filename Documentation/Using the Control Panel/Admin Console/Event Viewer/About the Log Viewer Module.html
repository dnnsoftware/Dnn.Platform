About the Event Viewer

The Admin > Event Viewer page displays the Log Viewer which provides an historical log of database events such as events which are scheduled, exceptions, account logins, module and page changes, user account activities, security role activities, etc. Authorized users can send exceptions to any email address. SuperUsers can add, edit and delete event records for all sites.

The Log Viewer can be deployed to sites and be added to site pages.
