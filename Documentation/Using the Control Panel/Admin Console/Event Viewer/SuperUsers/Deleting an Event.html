Deleting an Event

How to delete a logged event from the Log Viewer. Restricted to SuperUsers.

  1. Navigate to Admin >  Event Viewer - OR - Navigate to a Log Viewer module.
  2. Click the Edit Log Settings button located below all entries. This displays the Edit Log Settings page.
  3. Click the  Edit button beside the item to be deleted.
  4. Click the Delete button at the base of the module. This displays the message "Are You Sure You Wish To Delete This Item?"
  5. Click the Yes button to confirm.
