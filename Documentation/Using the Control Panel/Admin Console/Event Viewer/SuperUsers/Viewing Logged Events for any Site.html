Changing Log Viewer Records

How to filter the event records that are displayed in the Log Viewer by a single event type and set the number of records that are displayed on one page. Note: The records per page will default back to 25 records each time the page is refreshed.

  1. Navigate to Admin >  Event Viewer - OR - Navigate to a Log Viewer module.
  2. Expand the Viewer section.
  3. At Website, select All to view events from all sites - OR - Select the required site name.
  4. Optional. At Type, select an event type to filter events by that event type.
  5. Optional. Click on an event record to view full details of that event. Re-click the event record to close it.
  6. At Type, select an event type from the drop down list. The matching records will then be displayed below.
  7. Optional. At Records per page, select the number of records to be displayed per page from the drop down list.
