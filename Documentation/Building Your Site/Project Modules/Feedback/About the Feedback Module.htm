About Feedback

Feedback allows users to send messages to the site Administrator. Feedback Comments is a companion module that is added to the page whenever the Feedback module is added to a page. See "About Feedback Comments"

Installation Note: Feedback must be deployed and installed on your site by a SuperUser. See "Deploying and Installing More Extensions"

Module Version: 06.02.01 / Minimum DNN Version: 06.02.00

Features: ISearchable

Optional Settings:

  • Subject and category list management
  • Hide or display multiple form fields
  • Optional moderation for one or more feedback categories
  • Templates enable editors to change the module's look and feel
  • Set unique permissions to manage moderation and lists
  • Set validation rules for email, postal code, and telephone
  • Optional CAPTCHA security code
