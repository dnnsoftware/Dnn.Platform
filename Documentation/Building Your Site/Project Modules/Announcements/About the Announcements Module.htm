About Announcements

Announcements displays a list of announcements. Each announcement includes a title and a rich text description. Announcements are ordered by their publish date by default; however view order can be specified. Optional fields include an image, date of publication and a read more... link.

Installation Note: Announcements must be deployed and installed on your site by a SuperUser. See "Deploying and Installing More Extensions"

Module Version: 04.00.03 / Minimum DNN Version: DNN 04.07.00

Features: IPortable, ISearchable
