Deleting a Filtered Word

How to delete a filtered word from the Forum module.

  1. Go to the Forum Administration page. See "Navigating to the Forum Admin Page"
  2. In the left-hand panel, select Content > Filter Words. This displays the Filtered Word Management window.
  3. In the Filter column, locate the word to be deleted. See "Filtering Filter Words"
  4. Click the Delete  button.  This displays the message "Are You Sure You Wish To Delete This Item?"
