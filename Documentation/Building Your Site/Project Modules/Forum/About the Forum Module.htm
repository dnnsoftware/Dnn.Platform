About the Forum Module

The Forum module manages multiple discussion groups and forums. Each forum post includes a subject title and content. Optional settings include the ability to quote an existing post, enable post attachments, pinning a post, notification of replies posts, lock posts, as well as manage user settings and a wide range of Administration options.

Module Version: 05.00.03 / Minimum DNN Version: 06.00.00

Features: IPortable, ISearchable, IUpgradable
