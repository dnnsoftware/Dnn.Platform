Editing your Forum Profile

How to edit your user profile via the Forum module. Note: This tutorial explains how to manage your user profile when the Forum module uses the core DNN User Profile. Where an alternate non-core module is used, you must follow instructions provided by your module developer from Step 2.

  1. Select My Settings from the module actions menu - OR - Click the My Settings link. This opens the User Settings page. The Control Panel > Overview window is displayed.
  2. In the right-hand panel, select Profile > Edit Profile. This displays your User Profile for this site.
  3. Modify your profile as required. See "Managing your User Profile", See "Managing your User Account" and See "Managing your Profile Photo"
