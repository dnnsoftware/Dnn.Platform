Replying to a Post

How to reply to a post in the Forum module.

  1. Locate and open the required post. See "Viewing any Post"
  2. Click the Reply link located top right above the post. This opens the Add/Edit Post page which displays the post in the Editor.
  3. In the Editor, enter your post, any attachments and set optional fields as required. See "Adding a Forum Post (Advanced)"
  4. Optional. At Notification, if you wish to receive an email when the post has a response.
  5. Click the Submit link.
