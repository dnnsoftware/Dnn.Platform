Managing Forum Bookmarks

How to manage your thread bookmarks which you have chosen to track using the Forum module.

  1. Select My Settings from the module actions menu - OR - Click the My Settings link. This opens the User Settings page. The Control Panel > Overview window is displayed.
  2. In the right-hand panel, select Control Panel > Bookmarks. This displays Bookmarks window.
  3. You can perform these options:
    • Remove Bookmark: Click on the Delete button beside a bookmark
    • Go to Thread: In the Subject column, click on the [Thread Name] link
    • View Last Post: In the Last Post column, click on the [Date/Time] link
