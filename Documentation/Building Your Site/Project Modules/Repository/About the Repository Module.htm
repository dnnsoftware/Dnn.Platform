About the Repository

The Repository stores and displays a range of items. The type of items stored is controlled through the skin which is applied to the module. A wide range of skins is included allowing your to using the Repository for numerous purposes such as a simple blog, a media repository, a file repository, an articles repository, a compact file listing and more. The Repository offers fine grained permissions that allows you to control which roles can download, upload, rate and comment on items. Features such as moderation and personal are also included.

Installation Note: The Repository must be deployed and installed on your site by a SuperUser. See "Deploying and Installing More Extensions"

Important Upgrade Note. If you have modified any of the standard templates that are part of the Repository package, be sure to back them up before installing this new release, then restore them after the release has been installed. Alternatively, you can copy them to the /Portals/n/RepositoryTemplates folder which is the preferred location for storing modified templates.
