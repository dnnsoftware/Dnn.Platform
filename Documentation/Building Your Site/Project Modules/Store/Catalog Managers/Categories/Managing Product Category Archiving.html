Managing Product Category Archiving

Catalog Managers can enable or disable archiving of a product category on the Store module using the Store Admin module.

  1. On the Store Admin module, click the Categories link.
  2. Click the Edit link beside the category to be archived.
  3. At Archived, select from these options:
    • to archive the product. Archiving a category removes it and its associated products from the Store Menu module. Archiving a parent category, will archive all of its child categories.
    • to remove archiving. Restoring a category will display it in the Store Menu module. Restoring a parent category also restores all child categories unless they are individually set as archived.

  1. Repeat Steps 2-4 to modify archive status for additional categories.
