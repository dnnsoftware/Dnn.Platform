Editing a Blog Comment

How to edit a comment associated with a blog entry on the Blog module. This task can be performed by bloggers on their own blog or by the Administrators on all blogs.

  1. Locate the required blog entry. See "Viewing Blog Entries for a Month", See "Viewing Blog Entries for a Single Date", or See "Viewing Recent Entries for a Single Blog".
  2. Click the  Comments (1) link.


  1. Click the Edit  button or the Edit link beside the comment. The comment is now displayed in the Comment text box below.
  2. In the Comment text box, edit the comment.
