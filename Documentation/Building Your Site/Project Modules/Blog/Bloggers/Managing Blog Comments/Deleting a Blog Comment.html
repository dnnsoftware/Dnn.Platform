Deleting a Blog Comment

How to delete a comment associated with a blog entry in the Blog module. This task can be performed by bloggers on their own blog or by Administrators on all blogs.

  1. Locate the required blog entry. See "Viewing Blog Entries for a Month", See "Viewing Blog Entries for a Single Date", or See "Viewing Recent Entries for a Single Blog".
  2. Click the  Comments (1) link. This displays the comments associated with the selected entry. 

  1. Click the Delete button or the Delete link beside the comment to be deleted.
  2. Repeat Step 3 to delete additional comments from this blog entry.
