Adding Document Categories

Documents displays a list of downloadable documents that can be sorted by categories. These categories can only be created by a SuperUser via the Host > Lists page.

You can choose to either navigate directly to the Host > Lists page and add a parent list (See "Adding a Parent List") and then add one or more list entries (See "Adding a List Entry") or follow the below steps to navigate to the Lists page.

  1. Go to the Documents module.
  3. Click the List Editor link. This opens the Host > Lists page.
