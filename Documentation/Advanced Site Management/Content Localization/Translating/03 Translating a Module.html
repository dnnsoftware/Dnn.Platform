Translating a Module

How to translate module content into a language other than the default site language.

A translatable version of the module must exist. See "Changing Reference Modules to Translatable Modules"

  1. View the site in the culture that you will be creating the translation for. See "Viewing a Site in a Secondary Language"
  2. Select the edit option for the module. See "Editing Module Content"
  3. Translate text as required.
