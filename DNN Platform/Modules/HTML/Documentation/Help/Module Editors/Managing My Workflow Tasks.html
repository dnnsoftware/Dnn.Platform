Managing My Workflow Tasks (HTML Module)

How to view and update content that required reviewing before it can be published using the HTML module. This task requires that Workflow is NOT set to direct publish. See "Setting a Workflow"

  1. Select My Work from the HTML module actions menu. This displays the My Work page and a list of the content to be reviewed and published.
  2. Click on one of the listed [Page Name] links. This displays the selected module.

  1. You can now choose to edit, preview and/or publish the unpublished content (See "Publishing a Draft"), or you can rollback to a previous version (See "Rolling Back Content")
