delete-role

Permanently deletes the given DNN Security Role. You cannot delete the built-in DNN security roles of Administrator, RegisteredUser, Subscriber, or UnverifiedUser. WARNING: This is a permanent action and cannot be undone

Options

Flag Type Required Default Description
--id string Yes The ID of the security role to delete. This value is required. However, if you pass the id as the first argument after the command, you do not need to explicitly use the --id flag.

Permanently A DNN Security Role

get-role 11

Results

Successfully deleted role 'Public' (11)
