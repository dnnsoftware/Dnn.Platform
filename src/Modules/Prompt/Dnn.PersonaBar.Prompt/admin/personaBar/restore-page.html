restore-page

Restores a page from the DNN Recycle Bin.

Options

Flag Value Required Default Description
--id integer See Notes Explicitly specifies the Page ID to delete. Use of the flag name is not required. You can simply provide the ID value as the first argument. Required if --parentid and --name are not specified.
--name string See Notes Specifies the name (not title) of the page that should be restored. This can be combined with --parentid to target a page name with a specific Parent page. Required if --parentid and --name are not specified.
--parentid string See Notes When you want to restore all child pages of a certain page, use this flag and specify the parent's Page ID. Required if --id and --name are not specified.

Restore a Deleted Page By Page ID

restore-page 999 OR restore-page --id 999

Restore a Page With A Specific Page Name

restore-page --name "Page1"

Restore All Pages With A Common Parent Page

restore-page --parentid 999
