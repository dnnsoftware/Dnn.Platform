get-task

Retrieves details for the specified Scheduler Task. DNN refers to these as schedules or scheduler items. Prompt refers to them as tasks.

Options

Flag Type Required Default Description
--id integer Yes The Schedule ID for the item you want to retrieve. If you pass the ID as the first value after the command name, you do not need to explicitly use the --id flag name.

Get A Task

get-task 11 OR get-task --id 11
