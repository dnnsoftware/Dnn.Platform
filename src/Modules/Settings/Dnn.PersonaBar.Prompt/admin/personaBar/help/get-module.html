get-module

Retrieves details about a single module in a specified page

Options

Flag Value Required Default Description
--id integer Yes Explicitly specifies the Module ID of the module to retrieve. Use of the flag is not required. You can simply provide the ID value as the first argument.
--pageid integer Yes The Page ID of the page that contains the module.

Get Information on a Specific Module

The code below retrieves the details for a module whose Module ID is 345 on a page 48

get-module 359 --pageid 48

The following version is a more explicit version of the code above, but does the same thing on a page 48.

get-module --id 359 --pageid 48

Results

ModuleId : 359
Title : Navigation
ModuleName : Console
FriendlyName : Console
ModuleDefId : 102
TabModuleId : 48
AddedToPages : 42, 46, 47, 48
