list-modules

Retrieves a list of modules based on the search criteria

Options

FlagTypeRequiredDefaultDescription
--pageid int No When specified, the command will show modules from specified page only. Use of the flag is not required. You can simply provide the ID value as the first argument.
--name string No The name of the module to search for. This accepts the wildcard (*) character to do partial searches. The Module Name is not the same thing as the module's Friendly Name or the module's Title. To find the Module Name, list-modules on a page containing the module. Searches are case-insensitive
--title string No The title of the module to search for. This accepts wildcard (*) placeholders representing 0 or more characters. Searches are case-insensitive.
--deleted boolean When specified, the command will find all module instances in the portal that are in the Recycle Bin (if --deleted is true), or all module instances not in the Recycle Bin (if operate --deleted is false). If the flag is specified but no value is given, it will default to true. This flag may be used with --name and --title to further refine the results
--page int No 1 Page number to show records.
--max int No 10 Page Size for the page. Max is 500.

List Modules in the Current Portal

list-modules

List Modules on Specific Page

list-modules 72

List Modules Filtered by Module Name

This will return all XMod and XMod Pro modules in the current portal

list-modules --name XMod*

List Modules on a Specific Page Filtered by Module Name

This will return all XMod and XMod Pro modules on the page with a Page/TabId of 72

list-modules 72 --name XMod*

List All Modules Filtered on Name and Title

This will return all modules in the portal whose Module Name starts with Site and Title starts with Configure.

list-modules --title Configure* --name Site*
